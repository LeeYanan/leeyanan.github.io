Projects

Research on Collision Avoidance of the Tele-operation System Based on Position Speed Control algorithm2015 - Now

C/C++
In Progress

This project is about the control and manipulation of a tele-operation robot system. The operator (the master) manipulates the robot (the slave) through two Joysticks. The position speed control algorithm is used to reduce time delay and avoid the collision between the robot and its surroundings.

Object Recognition Based on Neural Network and Force Feedback2017 - Now

C#, Python(numpy, matplotlib, tersorflow)
In Progress

When the robot presses the test object, we can obtain a series of data consisting of the pressing depth and its corresponding reaction force. Each sample was divided into up to five sections, and each section was used to train its corresponding neural network model. Finally, five models were conbined to recognize objects.
